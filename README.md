## 💡 Introduction

Find your best developer in your region! Search by region and techs.

## 📝 Minimal Requirements

- NodeJs 12.x

## 🚀 Install

```sh
$ git clone https://github.com/gabriel-roque/finDEV-app/
```

## 📝 Configure WebApp

```sh
$ cp .env.example .env
$ expo start
```
